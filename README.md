# **UFCoin**

Trabalho da disciplina de Projeto Integrador IV.


## **Executar**
---

Instalar dependências:

```
$ python3 -m pip install -r requiriments.txt
```

Executar:

```
$ python3 . [port]
```

- `port`: Argumento opcional. Representa a porta a ser usada pelo servidor http. Valor padrão: 5000.
